

## Funkce

- Moderní a responsivní design
- Přehled sekcí: O mně, Projekty, Kontakt
- Interaktivní UI komponenty
- Implementace routing a dynamických prvků v Angularu

---

## Instalace a spuštění

1. Klonuj repo:

             git clone git@github.com:DanielOdstrcil/Portfolio_angular.git

	2.	Nainstaluj závislosti:

            npm install

	3.	Spusť projekt:

            ng serve

Projekt bude dostupný na: http://localhost:4200
